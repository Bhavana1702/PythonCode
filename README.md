# SparkConnect - Modern Dating App

A visually appealing, interactive dating app built with Python, Flask, and Docker. Features profile cards with images from Unsplash, swipe functionality, matching, and real-time statistics.

## Features

- **Attractive UI**: Modern design with gradient colors, animations, and responsive layout
- **Interactive Profiles**: Swipe right to like, left to pass, with visual feedback
- **Real Images**: Profile images fetched from Unsplash for a realistic feel
- **Match System**: Simulated matching algorithm with notifications
- **Statistics Dashboard**: Track your matches, profile views, and match rate
- **Mobile-Friendly**: Responsive design with touch/swipe support
- **Docker Support**: Easy deployment using Docker containers
